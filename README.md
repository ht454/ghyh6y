# Sherlook Game Platform - شير لوك

منصة ألعاب تفاعلية ذكية مع نظام توليد الأسئلة بالذكاء الاصطناعي.

## المميزات الرئيسية

### 🤖 نظام الذكاء الاصطناعي
- **توليد أسئلة تلقائي**: يستخدم OpenAI GPT لتوليد أسئلة فريدة
- **تجنب التكرار**: نظام ذكي لضمان عدم تكرار الأسئلة
- **صعوبة متدرجة**: أسئلة مناسبة لكل مستوى (200، 400، 600 نقطة)
- **صور تلقائية**: البحث التلقائي عن الصور المناسبة من Pexels

### 🎮 مميزات اللعبة
- **فئات متنوعة**: 14 فئة مختلفة (دول الخليج، تاريخ، جغرافيا، إسلامي، وأكثر)
- **فرق متعددة**: دعم فريقين مع إدارة النقاط
- **واجهة عربية**: تصميم كامل باللغة العربية مع دعم RTL
- **تصميم متجاوب**: يعمل على جميع الأجهزة

## التقنيات المستخدمة

- **Frontend**: React 18 + TypeScript + Vite
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **AI**: OpenAI GPT-3.5-turbo
- **Images**: Pexels API

## الإعداد والتشغيل

### 1. تثبيت المتطلبات
```bash
npm install
```

### 2. إعداد متغيرات البيئة
انسخ ملف `.env.example` إلى `.env` وأضف مفاتيح API:

```env
VITE_OPENAI_API_KEY=your_openai_api_key_here
VITE_PEXELS_API_KEY=your_pexels_api_key_here
```

### 3. الحصول على مفاتيح API

#### OpenAI API Key:
1. اذهب إلى [OpenAI Platform](https://platform.openai.com/)
2. أنشئ حساب أو سجل دخول
3. اذهب إلى API Keys
4. أنشئ مفتاح جديد

#### Pexels API Key:
1. اذهب إلى [Pexels API](https://www.pexels.com/api/)
2. أنشئ حساب مجاني
3. احصل على مفتاح API

### 4. تشغيل المشروع
```bash
npm run dev
```

## كيفية عمل نظام الذكاء الاصطناعي

### 1. توليد الأسئلة
- يرسل طلب إلى OpenAI مع تفاصيل الفئة والصعوبة
- يتجنب الأسئلة المستخدمة سابقاً
- يولد سؤال + إجابة + معلومة إضافية + وصف الصورة

### 2. البحث عن الصور
- يستخدم Pexels API للبحث عن صور مناسبة
- يختار صورة عشوائية من النتائج
- يحفظ رابط الصورة مع السؤال

### 3. تجنب التكرار
- يحفظ معرفات الأسئلة المستخدمة في localStorage
- يمرر قائمة الأسئلة المستخدمة للذكاء الاصطناعي
- يضمن عدم تكرار نفس السؤال
